Engagement Rate
Formula:
    engagement_rate = (engaged / human_count) * 100
    if human_count == 0, engagement_rate = 0

AI Assistant Traction / AI Rate

    if has_ai_interest exists:
        ai_sessions = sum(has_ai_interest) over human rows
    else:
        ai_sessions = count rows where service_name contains "AI"
    ai_rate = (ai_sessions / human_count) * 100
    if human_count == 0, ai_rate = 0
Description:
    Measures share of human activity showing AI assistant/service interest.

Opportunity Value

Top Market

Top Service
Formula:
    top_service = most frequent service_name among human rows
Description:
    Service receiving the highest current human activity.

Audience/Data Quality
Formula:
    quality = round((1 - mean(is_bot over all rows)) * 100)
Description:
    Higher value means lower bot/noise share in the active dataset.

Active Markets
Formula:
    active_markets = number of unique countries among human rows
Description:
    Count of markets represented in the current human traffic.

Risk Alerts
Formula:
    if risk_score exists:
        risk_alerts = count human rows where risk_score >= 70
    else if is_anomaly exists:
        risk_alerts = sum(is_anomaly) over human rows
    else:
        risk_alerts = 0
Description:
    Count of high-risk records requiring executive or operations attention.

8. POTENTIAL CUSTOMER MODEL FORMULAS
============================================================

Potential Customer Score
Formula:
    if session is bot:
        score = 0
    else:
        score = 0
        +35 if session has demo request
        +30 if session has contact request
        +25 if session has event signup
        +20 if session has AI assistant interest
        +15 if session has service page visit
        +10 if distinct pages >= 3
        +10 if session_number_for_ip > 1
        +5  if business-hours session
        -10 if session has HTTP error status >= 400
        score = clip(score, 0, 100)
Description:
    Main heuristic score estimating visitor sales intent.

Potential Customer Signal
Formula:
    if session is bot:
        signal = 0
    else signal = 1 when any condition is true:
        has demo request
        has contact request
        has event signup
        converted_to_lead is true
        is_warm_lead is true
        segment == "High-intent"
        has AI interest and distinct pages >= 3
        has service page visit and distinct pages >= 2
    otherwise:
        signal = 0
Description:
    Binary flag identifying likely potential customers.

Potential Customer Type
Formula:
    if bot: "Filtered Noise"
    else if demo request: "Demo Interested"
    else if contact request: "Contact Interested"
    else if event signup: "Event Interested"
    else if AI interest: "AI Assistant Interested"
    else if service interest: "Service Interested"
    else if segment == "High-intent": "High-Intent Visitor"
    else: "General Visitor"
Description:
    Human-readable reason/category for the potential customer.

Intent Score
Formula:
    intent_score =
        +35 if has demo request
        +25 if has event signup
        +20 if has AI interest
        +30 if has contact interest
    intent_score = min(100, total)
Description:
    Measures direct conversion intent.

Engagement Score
Formula:
    engagement_score =
        distinct_pages_session * 15
        +10 if repeat visitor/session_number_for_ip > 1
        +5 if business-hours session
    engagement_score = min(100, total)
Description:
    Measures session depth and repeated activity.

Conversion Score
Formula:
    conversion_score =
        +50 if has demo request
        +40 if has contact request
        +30 if has event signup
    conversion_score = min(100, total)
Description:
    Measures closeness to explicit lead conversion.

Service Value Score
Formula:
    service_value_score =
        +30 if has AI interest
        +20 if has service page visit
        +10 if distinct_pages_session >= 2
    service_value_score = min(100, total)
Description:
    Measures interest in valuable service areas.

Recency Score
Formula:
    recency_score = 50
Description:
    Placeholder score because generated data is considered within the active
    analysis range.

Conversion Stage
Formula:
    if uri in /scheduledemo.php, /contact.php, /event.php:
        "Demo / Event Request"
    else if uri in /ai-assistant.php, /contact.php:
        "AI / Contact Interest"
    else if is_service_page:
        "Service Interest"
    else:
        "Website Visit"
Description:
    Funnel stage used for funnel and drop-off views.

Funnel Stage Order
Formula:
    Website Visit = 1
    Service Interest = 2
    AI / Contact Interest = 3
    Demo / Event Request = 4
Description:
    Numeric order used to sort funnel stages.

Conversion Path
Formula:
    starts with "Homepage"
    add "Service Page" if has_service_interest
    add "AI Assistant" if has_ai_interest
    add "Demo Request" if has_demo_request
    else add "Contact" if has_contact_interest
    else add "Event Signup" if has_event_signup
Description:
    Text path describing the visitor journey.

Dropoff Stage
Formula:
    if converted_to_potential_customer:
        dropoff_stage = "Converted"
    else:
        dropoff_stage = conversion_stage
Description:
    Identifies where non-converted visitors dropped off.

Engaged Session
Formula:
    is_engaged_session =
        distinct_pages_session >= 3 OR request_count_session >= 4
Description:
    Session is engaged if it has enough page depth or request activity.

Session Depth Category
Formula:
    if distinct_pages <= 1: "Single page"
    if distinct_pages == 2: "2 pages"
    if distinct_pages <= 5: "3-5 pages"
    else: "6+ pages"
Description:
    Friendly bucket for session depth.

Session Duration Category
Formula:
    if duration_seconds < 60: "< 1 min"
    if duration_seconds < 300: "1-5 min"
    if duration_seconds < 900: "5-15 min"
    else: "15+ min"
Description:
    Friendly bucket for session duration.

Visitor Type
Formula:
    if is_bot: "Bot"
    else if session_number_for_ip == 1: "First-time visitor"
    else if session_number_for_ip > 3 and potential_customer_score >= 35:
        "Repeat high-intent visitor"
    else:
        "Returning visitor"
Description:
    Classifies visitor history and intent.

Campaign Impression Estimate
Formula:
    campaign_impression_estimate = int(campaign_cost_estimate / 0.05)
Description:
    Estimated impressions from campaign spend assuming 0.05 cost per impression.

Campaign Click Estimate
Formula:
    campaign_click_estimate = int(campaign_impression_estimate * 0.035)
Description:
    Estimated clicks assuming 3.5% click-through rate.

Campaign ROI Estimate
Formula:
    if campaign_cost_estimate > 0:
        campaign_roi_estimate =
            (potential_customer_score * 500 - campaign_cost_estimate)
            / max(campaign_cost_estimate, 1)
    else:
        campaign_roi_estimate = 0
    campaign_roi_estimate rounded to 2 decimals
Description:
    Heuristic ROI estimate from potential customer score and campaign spend.

Campaign Quality Score
Formula:
    campaign_quality_score = clip(campaign_roi_estimate * 10 + 50, 0, 100)
    rounded to 1 decimal
Description:
    Converts ROI estimate into a 0-100 campaign quality score.

Visit Share
Formula:
    visit_share = country_request_count / total_requests
    rounded to 4 decimals
Description:
    Share of all activity coming from each country.

Conversion Share
Formula:
    conversion_share = potential_customer_signal / max(total_potential_customer_signals, 1)
    rounded to 6 decimals
Description:
    Row-level contribution to total potential customer signal volume.

Promotion Gap Score
Formula:
    if is_service_page and not is_campaign_period:
        promotion_gap_score = random uniform value from 0.3 to 0.8, rounded to 2 decimals
    else:
        promotion_gap_score = 0.0
Description:
    Flags service pages that may be under-promoted.

Potential Customer Share Variance
Formula:
    potential_customer_share_variance =
        actual_potential_customer_share - target_potential_customer_share
    rounded to 4 decimals
Description:
    Difference between actual potential customer share and target share.

Potential Customer Target Ratio
Formula:
    pc_ratio = actual_potential_customer_share / max(target_potential_customer_share, 0.001)
Description:
    Used to classify market status and investment recommendation.

Market Status and Investment Recommendation
Formula:
    if pc_ratio >= 1.0:
        market_status = "Above Target"

        investment_recommendation = "Review"

        investment_recommendation = "Pause"

Market Growth Rate

    market_growth_rate = 0.08

Strategic Score

Description:
    Composite score combining potential customer share and traffic share.

Anomaly Category
Formula:
    if is_anomaly and anomaly name contains "bot" or "spike":
        "Traffic Spike"
    else if is_anomaly and name contains "broken" or "link":
        "High Error Rate"
    else if is_anomaly and name contains "viral":
        "Traffic Spike"
    else if is_anomaly:
        "Suspicious Pattern"
    else if status_code >= 500:
        "High Error Rate"
    else if response_time_ms > 1000:
        "Slow Response"
    else if is_bot:
        "Bot Noise"
    else:
        "Normal"
Description:
    Categorises operational anomalies and abnormal traffic.

Risk Score

    risk_score =
        +40 if is_anomaly

Description:
    Operational risk score for each row.

Risk Level

Requires Review

Data Quality Flag

Data Quality Score

Estimated Deal Value

12. DELTA / COMPARISON FORMULAS
============================================================

Count Delta
Formula:
    if baseline <= 0:
        delta_text = "starting"
        delta_class = "watch"
    else:
        diff = current - baseline
        delta_text = (diff / baseline * 100) formatted as signed percentage
        delta_class = "up" if diff >= 0 else "down"
Description:
    Used for count-based KPI comparisons against yesterday or 7-day average.

Rate Delta
Formula:
    if baseline <= 0:
        delta_text = "starting"
        delta_class = "watch"
    else:
        diff = current_rate - baseline_rate
        delta_text = diff formatted as signed percentage points
        delta_class = "up" if diff >= 0 else "down"
Description:
    Used for percentage/rate KPI comparisons.

Baseline Selection
Formula:
    if yesterday data exists:
        baseline = yesterday KPI stats
        label = "yesterday"
    else:
        baseline = mean KPI stats for prior 7 days
        label = "7-day avg"
Description:
    Provides comparison values for dashboard KPI deltas.
